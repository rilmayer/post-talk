「ぽすとーく」とは

LINEのおかけで、遠く離れた大切な家族とも気軽にコミュニケーションできるようになりました。でも、スマホやパソコンを使うのが難しい相手とはどうでしょうか。

電話はタイミングに気を使うし、手紙は手間がかかるし、LINEほど気軽には連絡できません。大切な相手には違いないのに、どうしても連絡の数が減ってしまうかも。

そこもつなげられるのが、ぽすとーく。

LINEを使えない相手にも、LINEからチャット感覚で手紙を郵送できます。

手紙が送られる仕組み

※ 一部機能は未実装です。

本件はオープンソースソフトウェアです。こまかい技術仕様や、ソースコードはGitHubでご確認いただけます。

使い方

  1. LINE BOT「ぽすとーく」と友達になります。こちらのリンクからご登録ください。

  2. 手紙を送るために初期設定を行います。チャットの指示に従って、自分と相手の情報を老特してください。

  3. メニューの「手紙を送る」を実行することで新しい手紙を作成します。手紙に印刷したいメッセージをチャットに入力してください。

  4. LINE Payで料金をお支払いください。料金は1通あたり300円です。

  5. 手紙が印刷され、相手に郵送されます。手紙は1週間前後で到着します。
