ぽすとーく とは

「LINE」のおかけで、遠く離れた大切な家族とも気軽にコミュニケーションできるようになりました。でも、スマホやパソコンを使うのが難しい相手とは？

電話はタイミングに気を使うし、手紙は手間がかかるし、「LINE」ほど気軽にはできません。大切な相手には違いないのに、どうしても連絡の数が減ってしまうかも。

そこもつなげるのが、ぽすとーく。

「LINE」を使えない相手にも、「LINE」からチャット感覚で手紙を郵送することができます。
